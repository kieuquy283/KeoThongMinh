# KeoBot Desktop

Electron shell for packaging the KeoBot voice MVP as a Windows desktop app.

## Status

This folder currently defines the desktop package skeleton. The main process,
backend lifecycle, and production packaging flow are implemented in later tasks.

## Scripts

```bash
npm run dev
npm run start
npm run pack
npm run build
```

## Environment

Use `desktop/.env.example` as the starting point for desktop-specific env vars.
