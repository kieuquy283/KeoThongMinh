# KeoBot Frontend

React + Vite frontend for the KeoBot voice pipeline MVP.

## Setup

```bash
cd frontend
npm install
```

## Run

```bash
npm run dev
```

## Checks

```bash
npm run typecheck
npm run build
```

## API Base URL

The frontend defaults to `http://localhost:8000`. Override with:

```env
VITE_API_BASE_URL=http://127.0.0.1:8000
```
